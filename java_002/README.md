1. 클래스 기초 개념 정리
    - 클래스와 객체의 정의
        - 클래스 : 추상화를 거쳐 사용할 데이터와 수행할 동작을 모아둔 설계도(틀)
        - 객체 : 클래스의 형식에 맞춰진 데이터들의 모음을 메모리에 저장한 것
    - 클래스 구성 요소
        - 변수부 / 생성자부 / 메소드부
            - 변수부 - 멤버변수들을 모아 둔 부분. 보통 private 로 두고 getter와 setter 를 이용해 값을 얻거나 저장한다. 이렇게하면 데이터의 변경을 제한할 수 있어 무결성을 지키기 쉽다.
                
                [접근제한자] 자료형 변수명;   // public + protected# [default]~ private -
                
            - 생성자부 - 생성자들을 모아 둔 부분. 아무것도 작성하지 않으면 기본 생성자를 제공한다. 객체를 생성할 때 초기화를 시켜주거나 생성할 때 꼭 필요한 멤버 변수를 지정할 수 있다.
                
                [접근제한자] 생성자명(매개변수){
                
                }    
                
                생성자명은 클래스명과 대소문자까지 완벽하게 같아야하며 매개변수 정보에 따라 오버로딩이 가능하다.
                
            - 메소드부 - 메소드들을 모아둔 부분. 메소드는 특정 행동을 수행하기 위한 코드 덩어리 이다. 
            [접근제한자] [예약어] 반환형 메소드명(매개 변수){
                // 실행할 내용
            }
            하나의 클래스 내에서 동일한 메소드명으로 여러개의 메소드를 만들어 메소드 오버로딩이 가능하나 매개변수의 정보(개수 타입 순서)가 달라야한다.
